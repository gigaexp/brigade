# Preset — React + MobX (strict)

Full opinionated rule set for React projects using MobX with presenter/container split,
strict i18n, and mandatory tests. Activate from your project's `CLAUDE.md`:

```markdown
@~/.claude/plugins/brigade/presets/react-mobx-strict.md
```
